发表于 2018-07-14

因为小学期是大数据培训，需要安装配置Hadoop，最近Docker比较火，所以打算尝试一下。

什么是Docker

先说一下什么是Linux容器吧，虚拟机因为资源占用多、启动慢等原因，但是虚拟机还原的是整个软件的原始环境，无法轻松应用于各个宿主机，所以Linux容器诞生了。Linux不是模拟的完整的操作系统，而是对线程的隔离（在正常的线程外套了一层保护壳），里面的进程访问的各个资源都是虚拟的，隔离了底层的系统。

Docker属于Linux容器的封装，提供简单易用的容器使用接口；Docker的操作其实和代码管理Git比较相似，用户可以方便的创建和使用容器，把自己的应用放入容器。容器还可以进行版本管理、复制、分享、修改。

Docker用途

  1. 提供一次性环境。比如，本地测试他人的软件、持续集成的时候提供单元测试和构件环境。
  2. 提供弹性的云服务。Docker 容器可以随开虽关，很适合动态扩容和缩容。
  3. 组件微服务架构。通过多个容器，一台机器可以跑多个服务，因此在本机可以模拟出微服务架构

Docker安装

本次试水的是windows上docker安装和应用，Linux以后有机会再尝试

  1. 在docker store中下载相应的windows版本安装
  2. 启用windows中Hyper-V功能，并且进入BIOS打开虚拟化
  3. docker for windows 安装，一路默认安装就可以，安装完成。
    成功后托盘里有一个鲸鱼图标，安装过程并未遇到任何坑
    并且可以用一些命令验证是否安装成功
